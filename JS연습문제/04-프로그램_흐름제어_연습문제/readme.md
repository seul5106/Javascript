## 문제1.

다음의 소스코드를 if문으로 재구성하시오.

```js
const 과학 = 72;

switch (과학 > 70) {
    case true:
        console.log("이 과목을 Pass 했습니다.");
        break;
    default:
        console.log("이 과목을 Pass하지 못했습니다.");
        break;
}
```
